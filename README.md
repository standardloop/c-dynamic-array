# c-dynamic-array

## Goals

- Create an array with a size, but be able to resize it if needed
- Allow *any* type to be inserted in the dynamic array
- No memory leaks

## Example
```sh
$ make
[
  [
    [
        [0,1,2,3,4,5],
        1000,
        "hello"
    ]
  ],
  [
    [0,1,2,3,4,5],
    1000,
    "hello"
  ]
]
```
